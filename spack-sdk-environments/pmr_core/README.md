# PMR Core

## Overview

This Spack environment contains the following software packages:

* Legion
* Kokkos
* RAJA
* CHAI
* PaRSEC
* DARMA
* GASNet-EX
* Qthreads
* BOLT
* UPC++
* MPICH
* Open MPI
* Umpire
* QUO
* Papyrus
* SICM
* AML


Other tools from the PMR Core  group are not (yet) included:

* upc
* sicm
* chai
* parsec
