# xSDK Stack

## Overview

This Spack environment contains the following software packages:

* hypre
* FleCSI
* MFEM
* Kokkoskernels
* Trilinos
* SUNDIALS
* PETSc/TAO
* libEnsemble
* STRUMPACK
* SuperLU
* SLATE
* MAGMA


Other tools from the xSDK  group are not (yet) included:

* ForTrilinos
* DTK (Trilinos option)
* Tasmanian
